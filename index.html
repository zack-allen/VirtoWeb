Generate full, runnable website projects from a single JSON schema

Author your site once with a unified JSON schema and produce self-hostable projects in PHP, Node (Express), Python (Flask) or plain static HTML/CSS/JS. VirtoWeb keeps templates and output predictable and dependency-light.

Get started Try the desktop builder Generator reference

Single source of truth

Pages, components, layouts, forms and assets described in a single JSON schema.

Multi-language outputs

Generate projects for PHP, Node.js (Express), Python (Flask), or static HTML with consistent structure.

Local-first tooling

Run everything locally — generator, CLI, and the desktop builder. No hosting required to generate code.

Quick commands

python -m venv .venv
.\.venv\Scripts\Activate.ps1  # Windows PowerShell
pip install -r generators/static_generator/requirements.txt
python -c "from generators.core.generator import generate; generate('examples/example_layout_site.json')"

Try it

Generate the example static site and open dist/static-example/index.html.
